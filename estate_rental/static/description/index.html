This is a rental Management module

This module provide feature to manage rental sales to customers.


Video Tutorial: Click Here to Watch


- For configuration: Go to Rental Menu, Under configuration, Click on Set Rental Products and create a product .


- Tick the 'Is rent' checkbox then, select the 'Unit Available', and set the sales price per year.


- The create a new rental; Add customers and then add rental items.



{% comment %} - When you create an Invoice for the sales order, you will also see the sales commission passed to the commission line in the invoice page.
- You can also validate this to create or update an existing worksheet.
- When you want to register a payment for the transaction, please Check on the 'Apply Commission'checkbox to indicate to you want to include commission for the payment
Commission will be used to calculate the commission payment.

- Once done, the system automatically updates(Add) the paid commission to the 'Amount paid' field of the existing commission worksheet.
- Once the value of total paid commission is equal to the commission 'Amount to pay', the system marks the status to paid completely.
- You can also go to the worksheet under Sales Commission Menu to register or create commission payment any time you want.
{% endcomment %}
